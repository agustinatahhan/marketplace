pasos para crear el repositorio local

git init // crea el repositorio
git config user.name "nombreusuario" //agrega nuestra identidad
git config user.email "nombreusuario@gmail.com" //agrega nuestro email
git remote add origin https://github.com/DH/RepoRemoto //apunta al repositorio remoto

git remote -v para verificar que esta todo bien


**************************************************************

1. Clonar el Repositorio:  git clone https://github.com/tu-usuario/nombre-repositorio.git (esto se hace una sola vez al principio)

2. Actualiza la Rama Principal:  git pull origin main

3. Crear nueva rama: git checkout -b nombre-rama (cada uno de nosotros trabajara sobre una rama distinta y cada cambio que hagan lo hacemos sobre esa misma rama)
4. Verifica que estás en la nueva rama con: git branch
5. Agrega y Confirma los Cambios 
git add .
y luego
git commit -m "Mensaje de la confirmación"

6. Sube la Rama al Repositorio Remoto 
git push origin nombre-rama

7. Crea una solicitud de extracción En GitHub haciendo clic en Pull Request (Aquí seleccionarán la rama que han creado y que desean fusionar con la rama principal)
8. Fusionar la solicitud de extracción:  el propietario del repositorio puede fusionar la solicitud de extracción
9. Seleccionar la Solicitud de Extracción:  Haz clic en la pestaña "Pull requests"
10. Fusionar Cambios: haz clic en el botón "Merge pull request".
11. Confirmar la Fusión:Haz clic en el botón +"Confirm merge"
